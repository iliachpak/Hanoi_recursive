# 🗼 Hanoi Visualizer

A simple recursive visualization of the Tower of Hanoi problem.

---

## 🚀 Run the project

```bash
make
./hanoi [nbr_of_disks]
```

Example:

```bash
./hanoi 8
```

---

## 🎬 Controls

- The animation runs automatically in real-time
- To stop the program:

```
Ctrl + C
```

---

## 🧠 Visualization

The program displays the Tower of Hanoi state in the terminal:

```
         |                  |                  |
         |                  |                  |
      =======               |                  |
     =========              |                  |
    ===========             |                  |
   =============            |                  |
  ===============           |                  |
 =================         ===               =====
#########################################################
```

Each frame represents a step of the recursive solution.
